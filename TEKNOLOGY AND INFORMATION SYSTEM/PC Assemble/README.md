#### PC ASSEMBLE
#### DATE : 25 NOVEMBER 2025

This project documents the process of assembling a custom PC from individual hardware components, including CPU, motherboard, RAM, storage, GPU, and power supply installation.
The project focuses on understanding hardware compatibility, system integration, and basic troubleshooting, serving as a practical learning experience in computer hardware and system architecture.
